Micro Campeonato Regional

Designed and delivered a full stack tournament ecosystem, including infrastructure automation and reliable deployment workflows.

Scope

  • Web app for tournament management and results.
  • API platform for teams, fixtures, and standings.
  • Infrastructure as code on AWS with Terraform.

Tech stack

  • Vue 3 frontend with optimized UI flows.
  • LoopBack API for data access and security.
  • Terraform + Docker for repeatable deployments.

Outcomes

  • Reliable deployment pipeline with infrastructure reproducibility.
  • Centralized data model with automated backups.
  • Improved operational visibility with dashboards and alerts.
